# 납품 패키지 구성안

이 문서는 `반도체 장비 디지털 트윈 교육 시뮬레이터`를 학교, 교육기관, 기업 PoC에 납품할 때 포함할 수 있는 산출물과 범위를 정리합니다.

## 기본 패키지

- Unity 원본 프로젝트
- 실행 가능한 데모 빌드
- 샘플 디지털 트윈 씬
- 6축 로봇 조그, 티칭, 재생 기능
- 웨이퍼 이송 시나리오
- 충돌 감지 기반 E-STOP 안전 모니터
- 설치 및 실행 안내
- 강사용 수업 운영 가이드
- 학생용 실습 과제

## 교육 패키지 예시

- 2시간 소개 강의: 반도체 장비 자동화와 디지털 트윈 개념
- 4시간 실습 과정: Unity 씬 실행, 로봇 조그, 웨이퍼 이송, E-STOP 확인
- 8시간 심화 과정: 장비 시나리오 수정, UI 개선, 안전 인터락 확장
- 프로젝트형 과제: 신규 장비 모듈 또는 교육용 시나리오 추가

## 커스터마이징 범위

- 기관명 또는 교육과정에 맞춘 표지, 메뉴, 안내 문구 적용
- 장비 배치, FOUP, 챔버, 로봇 위치 등 씬 구성 변경
- 수업 난이도에 맞춘 실습 단계와 평가 문항 작성
- 특정 장비 공정 흐름을 반영한 시나리오 추가
- 실행 파일 빌드 및 배포 패키지 구성

## 납품 전 점검 항목

- 지정 Unity 버전에서 프로젝트가 정상 열리는지 확인
- 콘솔 컴파일 에러가 없는지 확인
- 샘플 씬에서 로봇 조그 UI가 동작하는지 확인
- 웨이퍼 이송 시나리오가 재생되는지 확인
- 충돌 발생 시 E-STOP이 발동하는지 확인
- 교육용 PC 권장 사양과 설치 절차를 문서화

## 수익화 방향

오픈소스 저장소는 신뢰 확보와 기술 검증을 위한 공개 버전으로 운영하고, 실제 매출은 교육 운영, 기관 맞춤 커스터마이징, 유지보수, 강의 자료 제작, 현장 워크숍에서 만드는 방향을 권장합니다.
